# 医院预约系统-后端
注意，请在dev分支上开发，**不要随便合并 main**

## 开发环境
- 开发工具 IDEA 2021.2.3
- 开发环境 JDK 11
- maven

## 技术和框架

- 项目构建：maven
- web框架：spring boot
- 数据库ORM：mybatis-plus
- 数据库：MySQL
- 文档接口工具：swagger2
